# 7. Best Time to Buy and Sell Stock

## Sliding Window
```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        maxprofit = 0
        l = 0
        r = 1
        while r < len(prices):
            if prices[l] > prices[r]:
                l = r
                r += 1
            else:
                profit = prices[r] - prices[l]
                maxprofit = max(maxprofit, profit)
                r += 1
        return maxprofit
```
Time Complexity: O(n)
Space Complexity: O(1)

Most Important: l = r
